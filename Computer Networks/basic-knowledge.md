# 计算机网络基础概念
* 互联网服务提供商 (ISP )：指向用户提供互联网接入服务的公司或机构。
* 节点 (Node)：网络中连接点或端点的统称。如计算机、交换机 (Switch)、路由器 (Router) 等构成网络的硬件。
* 主机 (Host)：也称为服务器 (Server)，是指通过网络为其它机器提供服务的计算机。
* 客户端 (Client)：指从主机处获得服务的计算机，也称为终端 (Terminal)。

# 速率与带宽
### 传输速率
* **定义**：每秒钟传输多少个比特 (bit)。
* **基本单位**：**bit/s** (或者 **b/s**, **bps** - bit per second)。
* **换算关系 (重要！网络中以 $10^3$ 为进位)**：
    * $1 \text{ kb/s} = 10^3 \text{ b/s} = 1000 \text{ b/s}$
    * $1 \text{ Mb/s} = 10^3 \text{ kb/s} = 10^6 \text{ b/s}$
    * $1 \text{ Gb/s} = 10^3 \text{ Mb/s} = 10^9 \text{ b/s}$
    * $1 \text{ Tb/s} = 10^3 \text{ Gb/s} = 10^{12} \text{ b/s}$
* **1 Byte (字节) = 8 bit (比特)**
* bit (比特)：计算机中最小的数据单位，代表一个二进制数 0 或 1。

### 网络带宽
* **定义**：在计算机网络中，带宽用来表示网络中某信道传送数据的**能力**。
* **物理意义**：表示在单位时间内网络中的某信道所能通过的“最高数据率”。
* **单位**：与速率一致，常用 **bit/s** 或 **b/s**。

## 易混淆点提醒

1. **带宽 vs 实际速率**：
   * **带宽**是这条路的“最高限速”或“车道宽度”（理论最大值）。
   * **传输速率**是此时此刻路上车流的“实际速度”。

2. **$1024$ 还是 $1000$？**
   * **存储容量** (如硬盘、内存)：通常使用 $1024$ 进位 ($2^{10}$)。
   * **传输速率** (如网速、带宽)：通常使用 $1000$ 进位 ($10^3$)。

3. **下载速度计算**：
   如果你的宽带是 $100\text{ Mbps}$，理论上的最高下载速度是 $100 \div 8 = 12.5\text{ MB/s}$。

## 物理层

* 屏蔽双绞线-STP
* 非屏蔽双绞线-UTP

2.光纤-fiber-F
光纤传输系统：光源+传输介质+检测器
* 单模光纤：直线传输无反射，衰减少，长距传输
* 多模光纤：全反射

3.同轴电缆-coaxial cable-C
屏蔽特性好，带宽更大

### 无线传输
* 线电波
* 微波：卫星通讯
* 红外线
* 激光

### 传输标准命名规则
* 1000BASE-TX
* 传输速率+调制方法+传输媒介+编码体系
* BASE:基带信号，一根线缆只传输一个信号
* BROAD：宽频信号，一根线缆可传输多个信号

### 编码
* 归零编码RZ：正电平代表逻辑1，负电平代表逻辑0，每传输完一位数据，信号返回0电平。
* 不归零编码NRZ：高的全是1，低的全是0
* 反向不归零编码NRZI：信号翻转表示0，不翻转表示1
* 曼彻斯特编码：由低到高为0，高到低为1
* 差分曼彻斯特编码：中间虚线看，有变化为0，无则为1

### 数字调制
* 数字调试：比特与代表它们的信号之间的映射过程
* 码元：数字调制过程中用来承载信息的基本信号单位
* 波特率：单位时间内传输的码元个数，单位为baud波特，可表示信号传输速率
* 比特率：单位时间内传输的比特个数/多少位二进制，单位为bit比特，可表示数据传输速率
* 波特率关注物理信号变化，即底层通信速率
* 比特率关注实际传输的数据量，即有用信息的速率
### 数字调制中波特率B与比特率C的关系
$C = B \log_2 N$
N为码元离散状态数，每个码元能表示的不同状态数量
### 数字调制方法
* 将数字信号调制为模拟信号有三种方式：调幅，调频，调相
* QAM正交振幅调制：在频率相同的条件下，将调幅与调相结合起来的调制技术，形成叠加的复合信号。
* QMA所用公式：$$C = B \log_2 (M \times N)$$，M相位数，N振幅数

### 奈奎斯特定理-无噪声极限（背公式）
* 无噪声（理想状态下），限制速率的唯一因素是带宽。 $$B = 2H$$，$B$ (Baud Rate)：码元速率极限值（波特率），$H$ (Bandwidth)：信道带宽。
* 无噪声信道传输速率：$$C = 2H \log_2 N$$
### 香农公式-有噪声极限（背公式）
* 受限于带宽和噪声的信道中，能够实现的最大信息传输速率（信道容量）。核心公式$$C = H \log_2(1 + S/N)$$，S信号功率，N噪声功率，S/N信噪比
* 知道分贝dB求信噪比S/N：$$\text{dB} = 10 \lg (S / N)$$
### 电路交换、报文交换、分组交换
* 电路交换：源站点与目标站点之间建立一条直通独占的物理通道。三个步骤：建立连接，传送数据，释放数据。
* 报文交换：发送前无需建立连接，以报文作为数据传输单位。
* 分组交换：切分：将大报文切成小分组。加头：每个分组加上首部（含地址和编号）。转发：路由器收到分组，先存入缓存，查表后再发出

## 信道复用
* 频分复用FDM：将宽带划分为多个频段，每个信号占据不同的频段进行传输。如收音机，广播。
* 时分复用TMD：多个信号按照时间片轮流占用整个信道，每个信号在不停的时间间隔内传输。如对讲机。
* 码分多路复用CDM：又称码分多址CDMA，所有用户在相同频率和时间共享信道，但每个用户使用不同的编码来区分信号。码片：一比特会被再细分成多个码片。任何码片与自身归一化内积一定为1。
接收到的码片*某节点码片序列，+1表示比特1，-1表示比特0，0表示无数据
* 波分复用WDM：类似于FDM，但用于光纤通信，每个信号用不同波长的光进行传输。

## 题型：

## 一、利用“四大公式”求事件概率

* 加法公式：

$P(A \cup B) = P(A) + P(B) - P(AB)$  
AUB表示AB至少发生一个

  $P(A \cup B \cup C)=P(A) + P(B) + P(C) - P(AB) - P(BC) - P(AC) + P(ABC)$  
  加积减偶

* 减法公式：

$P(A - B) = P(A) - P(AB)=P(A-B) = P(A) - P(B)$  
A-B表示A发生时B不发生

$P(\bar{A}) = 1 - P(A)$

* 乘法公式：

$P(AB) = P(A)P(B|A) \quad (P(A) > 0)$

* 条件公式：

$P(A|B) = \frac{P(AB)}{P(B)} \quad (P(B) > 0)$

---
### 事件独立性
如果事件 $A$ 的发生不影响事件 $B$ 发生的概率，反之亦然，则称 $A$ 与 $B$ 相互独立。
* $$P(AB) = P(A)P(B)$$
* $$P(B|A) = P(B)$$
* 若 $A$ 与 $B$ 独立则它们的对立事件间也相互独立。
## 二、古典概型求概率
$$P(A) = \frac{m}{n} = \frac{\text{事件 } A \text{ 包含的基本事件数}}{\text{样本空间包含的基本事件总数}}$$

## 三、全概率公式和贝叶斯公式
应用题中，若实验分两个阶段完成，则用全概率或贝叶斯公式完成。
* 全概率公式：事件 $A$ 发生的总概率，等于它在各种不同背景（划分）下发生概率的加权平均。
$$P(A) = \sum_{i=1}^{n} P(B_i)P(A|B_i)$$
* 贝叶斯公式：已知结果 $A$ 已经发生，反推它是由于某个特定原因 $B_k$ 造成的概率。
$$P(B_k|A) = \frac{P(B_k)P(A|B_k)}{P(A)} = \frac{P(B_k)P(A|B_k)}{\sum_{i=1}^{n} P(B_i)P(A|B_i)}$$
## 四、伯努利概型求概率
当出现“独立”，“N重试验”，“每次试验只有A和1-A两种结果，则用伯努利概型。
* 二项概率公式 $$P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, 2, \dots, n$$  

# 高数基础
### 1. 核心求导公式
应用场景：已知分布函数 $F(x)$，求概率密度函数 $f(x)$ 时使用。
| 函数类型 | 原函数 $F(x)$ | 导数 $f(x) = F'(x)$ | 概率论中的常见例子 |
| :---| :--- | :--- | :--- |
| **常数** | $C$ | $0$ | $(1)' = 0$ |
| **幂函数**  | $x^n$ | $n \cdot x^{n-1}$ | $(x^2)' = 2x$ |
| **对数函数**  | $\ln x$ | $\frac{1}{x}$ | $(\ln x)' = \frac{1}{x}$ |
| **指数函数**  | $e^{ax}$ | $a \cdot e^{ax}$ | $(e^{-2x})' = -2e^{-2x}$ |
### 2. 核心积分公式
应用场景：积分是求导的逆运算。已知密度函数 $f(x)$，需要求出它的原函数，以便进一步计算概率。
| 函数类型 | 待积函数 $f(x)$ | 积分结果（原函数） $\int f(x) dx$ |
| :--- | :--- | :--- |
| **常数** | $k$ | $kx$ |
| **幂函数** | $x^n$ ($n \neq -1$) | $\frac{1}{n+1} x^{n+1}$ |
| **幂函数特例** | $\frac{1}{x}$ | $\ln x$ |
| **指数函数** | $e^{ax}$ | $\frac{1}{a} e^{ax}$ |
### 3. 计算具体概率的核心法则
应用场景：计算变量落在某个区间的概率（即求曲线下方的面积）。
* 牛顿-莱布尼茨公式
$$\int_a^b f(x) dx = F(x) \Big|_a^b = F(b) - F(a)$$
### 4. 概率论专属的两个“计算特例”
在处理连续型随机变量（特别是正态分布和指数分布）时，你会高频遇到以下两种情况：  
* 特例一：处理无穷大在概率论中，积分的上限经常是正无穷 $+\infty$。你只需要记住指数函数在无穷远处的衰减特性：  
当 $a > 0$ 时，$e^{-\infty} = 0$。  
任何数的 $0$ 次方等于 $1$，即 $e^0 = 1$。
* 特例二：计算数学期望时的工具应用场景：  
求连续型变量的期望 $E(X) = \int x \cdot f(x) dx$ 时，通常会出现多项式乘以指数函数的情况（例如 $x \cdot e^{-2x}$）。这类积分无法直接查表，需要用到分部积分法。  
公式：$\int u \, dv = uv - \int v \, du$  
分部积分法：一种微积分技巧，将两个函数乘积的复杂积分，转化为另一个相对容易计算的积分。

## 三、连续型R.V.相关计算
### 连续型随机变量的关键性质
* 单点概率为零：$P\{X = a\} = 0$。  
* 对于连续型随机变量，概率密度函数 $f(x)$ 是 分布函数 $F(x)$ 的导数。  
$f(x) = F'(x)$
### 区间概率的计算
* 利用分布函数（最常用）：  
$P\{a < X \le b\} = F(b) - F(a)$  
直接将上限和下限代入分布函数相减即可。  
* 利用概率密度函数（积分法）：  
$P\{a < X \le b\} = \int_{a}^{b} f(x) dx$  
计算密度函数在对应区间上的定积分。  
* 累积定义：
$P\{X \le a\} = F(a) = \int_{-\infty}^{a} f(x) dx$  
这说明分布函数本身就是从负无穷到该点的概率累加。

### 知道概率密度求分布函数
* 无论密度函数长什么样，随机变量在整个实数轴上取值的总概率必须是 100%（即曲线下方的总面积恒等于 1）。  
$\int_{-\infty}^{+\infty} f(x) dx = 1$

## 四、常见的连续型分布计算
### 均匀分布 (Uniform Distribution)
变量在某个区间内的取值概率是“等可能的”。  
* 记法：$X \sim U(a, b)$，表示随机变量 $X$ 在区间 $(a, b)$ 上服从均匀分布。
* 概率密度函数：$$f(x) = \begin{cases} \frac{1}{b-a}, & a < x \le b \\ 0, & \text{其他} \end{cases}$$  
* 如果要求 $X$ 落在子区间 $[c, d]$ 内的概率，直接用目标区间的长度除以总区间的长度即可：$$P\{c < X < d\} = \frac{d - c}{b - a}$$（前提是 $[c, d]$ 完全包含在 $[a, b]$ 内部）。
### 正态分布 (Normal Distribution)
图形呈现“中间高、两边低”的钟形曲线。
* 记法：$X \sim N(\mu, \sigma^2)$。$\mu$ 是期望（决定中心位置），$\sigma^2$ 是方差（决定曲线胖瘦）。
* 正态曲线关于 $x = \mu$ 轴对称。这意味着以中轴线为界，左右两边概率相等均为一半：$$P\{X \le \mu\} = P\{X \ge \mu\} = 0.5$$
* $f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
